
ВОПРОС НОМЕР 1 -->


 Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделанно предварительно?

ОТВЕТ:

В окне Клонировать репозиторий ввести URL-адрес клона удаленного репозитория Git, который требуется клонировать, проверить путь к локальной папке, в которой  хотим создать локальный клон, а затем выбираем клонировать.



ВОПРОС НОМЕР 2 -->


Как создать git репозиторий из локального каталога на своем комьютере. Как его отправить на github.com?

ОТВЕТ:

Git репозиторий можно создать в следствии данных действий - создаем файл находясь нашей директории git  " > file.txt " затем мы добавляем через git add file.txt


ВОПРОС НОМЕР 3 -->

Как получить на локальный компьютер изменения с github.com?

ОТВЕТ:

Изменения можно получить с помощью команды git pull


Дополнительно 



Какую информацию нам дают команда :

git status - Просмотреть список новых или измененных файлов, которые еще не закомитены!

git log - Показать полную историю изменений

git remote -v - Верифицируем что удаленный репозиторий связан с нашим
